# 🌟 철학적 동반자 - 삶을 바꾸는 미연시 게임

## 🎯 프로젝트 비전

**"게임을 통해 무기력에서 벗어나 철학적 삶으로 인도하는 것"**

**철학적 동반자**는 단순한 미연시 게임이 아닙니다. 무기력한 사람들이 캐릭터와의 만남을 통해 자신에게 맞는 철학적 가치관을 발견하고, 실제 삶의 변화를 만들어가도록 돕는 것이 목표입니다.

## 🤝 핵심 개념: 바디 더블링

무기력한 사람에게 필요한 것은 조언이 아닌 **존재** 그 자체입니다. 누군가 곁에 있다는 것만으로도 힘이 되는 '바디 더블링' 개념을 게임에 구현했습니다.

### 🎯 핵심 컨셉
- **완전 익명제**: 개인정보 수집 없음, 로컬 스토리지만 사용
- **미연시 스타일**: Day 1-10 구조의 완전한 스토리 진행
- **4지 선택지 시스템**: 철학적 가치관에 따른 친밀도 등락
- **4가지 엔딩 분기**: 선택에 따른 다양한 결말
- **꿈과 현실의 경계**: 인셉션 스타일의 애매한 현실감
- **무한 재플레이**: 다른 선택으로 새로운 엔딩 도전

## 🎮 게임 플레이 구조

### Day 1-10 스토리 진행
```
📖 프롤로그: 밤늦게 발견한 신비로운 24시간 도서관
📅 Day 1-5: 각 캐릭터와 순차적 첫 만남
📅 Day 6-8: 친밀도에 따른 심화 관계 선택
📅 Day 9-10: 클라이맥스 및 엔딩 분기
☀️ 에필로그: 현실로 돌아온 후... 꿈이었을까?
```

### 🏗️ 배경 설정: 24시간 복합 도서관
- **1층**: 정보데스크, 북카페, 어린이코너
- **2층**: 힐링라운지, 열람실, 스터디룸  
- **별관**: 호수 산책로와 관리사무실

## 🎭 캐릭터 시스템

### 5명의 철학적 동반자들

#### 🗿 **최서진** (스토아 철학 - 마르쿠스 아우렐리우스 모티브)
- **위치**: 1층 정보데스크 (대학생 알바)
- **성격**: 차분하고 현실적
- **모에 포인트**: 어려도 어른스러운 당찬 아이
- **철학**: 통제 가능한 것에 집중, 흔들리지 않는 마음

#### 🔥 **강하연** (실존주의 - 니체 모티브)  
- **위치**: 1층 북카페 (츤데레 바리스타)
- **성격**: 까칠하지만 속정 있는
- **모에 포인트**: 츤데레 (말과 행동이 반대)
- **철학**: 의지력과 자기극복, 강인한 정신력

#### 🔍 **한지우** (고전 철학 - 소크라테스 모티브)
- **위치**: 1층 어린이코너 (독서광 어린이)  
- **성격**: 순수하고 호기심 가득
- **모에 포인트**: 똑똑한 막내 (끝없는 질문)
- **철학**: "무지의 지", 질문하는 것의 중요성

#### 🪷 **임수인** (불교 철학 - 부처 모티브)
- **위치**: 2층 힐링라운지 (자원봉사자)
- **성격**: 따뜻하고 포용적  
- **모에 포인트**: 치유의 큰누나 (무조건적 수용)
- **철학**: 고통의 수용, 현재 순간에 집중

#### 🌊 **서유진** (도교 철학 - 장자 모티브)
- **위치**: 별관 호수 관리소 (자유로운 관리인)
- **성격**: 자유롭고 편안함
- **모에 포인트**: 편안한 소꿉친구 (부담 없는 관계)  
- **철학**: 무위자연, 흘러가는 대로 순응

## 💖 친밀도 시스템 (0-20단계)

### 친밀도별 관계 단계
- **0-4**: 가까워지는 중 - 서로 알아가는 단계
- **5-9**: 관계 심화 - 더 깊은 대화와 이해
- **10-14**: 특별한 친구 - 언제든 만날 수 있는 관계
- **15-19**: 진정한 동반자 - 깊은 철학적 교감
- **20**: 완벽한 관계 - 특별 엔딩 조건

### 선택지별 친밀도 변화
- **최고 선택** (+2~+3): 캐릭터의 철학과 완벽히 부합
- **좋은 선택** (+1): 긍정적이지만 평범한 반응
- **보통 선택** (0): 무난하지만 특별하지 않은 반응
- **아쉬운 선택** (-1): 캐릭터가 실망하는 반응
- **최악 선택** (-2): 관계에 악영향을 주는 반응

## 🎯 4가지 엔딩 시스템

### ✨ **진정한 동반자 엔딩**
- **조건**: 한 명 이상 친밀도 20 달성
- **특징**: 현실에서 특별 아이템 발견 (꿈의 증거)
- **의미**: 한 사람과의 깊은 유대

### 🌟 **완벽한 동반자들 엔딩** 
- **조건**: 3명 이상 친밀도 15+ 달성
- **특징**: 모든 캐릭터들의 따뜻한 메시지
- **의미**: 여러 사람과의 균형잡힌 관계

### 🌱 **새로운 시작 엔딩**
- **조건**: 평균 친밀도 10+ 달성  
- **특징**: 성장과 변화에 대한 깨달음
- **의미**: 작지만 확실한 변화

### 🌙 **조용한 밤의 끝**
- **조건**: 그 외 모든 경우
- **특징**: 평범하지만 위로가 되는 경험
- **의미**: 재플레이 유도, 다른 선택 권유

## 🎨 특별한 연출 요소

### 꿈과 현실의 애매한 경계
```
🌙 게임 중: "정말 신비로운 도서관이네..."
☀️ 현실 복귀: "어? 이 아이템은 어디서 온 거지?"
❓ 의문점들: 시간, 장소, 상황의 비논리성
💭 영원한 토론: "진짜였을까, 꿈이었을까?"
```

### 친밀도에 따른 현실 증거
- **친밀도 20**: 특별 아이템이 실제로 존재
- **친밀도 15+**: 도서관 이용 카드 발견
- **친밀도 10+**: 모르는 번호의 안부 메시지
- **그 외**: 아무것도 없는 평범한 아침

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Storage**: LocalStorage API
- **Architecture**: Single Page Application (SPA)  
- **Design**: Glass morphism + Visual Novel UI
- **Dependencies**: 없음 (순수 바닐라 JS)

## 🚀 실행 방법

1. **간단 실행**: `index.html` 파일을 웹 브라우저에서 열기
2. **인터넷**: 연결 필요 없음 (완전 오프라인)
3. **브라우저**: 모든 최신 브라우저 지원 (Chrome, Firefox, Safari, Edge)
4. **저장**: 게임 진행은 자동으로 로컬에 저장

## 🎨 UI/UX 특징

### 미연시 스타일 디자인

- **친밀도 시각화**: 하트 아이콘으로 관계 진행도 표시

## 🎪 게임 플레이 팁

### 첫 플레이어를 위한 조언
- **솔직한 선택**: 자신의 마음에 솔직한 선택을 하세요
- **관계 집중**: 한 명과 깊은 관계를 맺는 것도 좋은 전략
- **재플레이 고려**: 첫 번째는 자연스럽게, 두 번째는 전략적으로

### 완벽 공략을 원한다면
- **균형잡힌 선택**: 모든 캐릭터에게 긍정적인 선택
- **철학 이해**: 각 캐릭터의 철학을 이해하고 그에 맞는 선택
- **인내심**: 20까지 친밀도를 올리는 것은 여러 번의 만남이 필요
