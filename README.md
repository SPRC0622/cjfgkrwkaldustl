# 🌟 철학적 동반자 - 웹 기반 철학 캐릭터 상호작용 시스템

## 📋 프로젝트 개요

**철학적 동반자**는 NEET/무기력 상태의 사람들을 위한 웹 기반 철학적 위로 및 동기부여 애플리케이션입니다. 5명의 철학자 모티브 캐릭터와의 상호작용을 통해 사용자가 혼자가 아님을 느끼고, 점진적으로 친밀도를 쌓아가며 철학적 지혜를 얻을 수 있도록 설계되었습니다.

### 🎯 핵심 컨셉
- **완전 익명제**: 개인정보 수집 없음, 로컬 스토리지만 사용
- **점진적 친밀도 시스템**: 미연시 스타일의 캐릭터 상호작용
- **랜덤 만남 + 개별 관계**: 처음엔 랜덤으로만 만나다가 친해지면 언제든 만날 수 있음
- **안전한 메시지 시스템**: 연락처/개인정보 자동 필터링
- **철학적 위로**: 각 캐릭터는 서로 다른 철학적 관점 제공

## 🎭 캐릭터 시스템

### 5명의 주요 캐릭터
1. **🪷 바다 (부처 모티브)** - 마음의 친구
   - 평온하고 따뜻한 성격, 불교 철학 기반
   - 고통의 수용과 현재 순간에 집중하는 지혜 제공

2. **🔍 퀘스트 (소크라테스 모티브)** - 호기심 많은 친구  
   - 질문하는 것을 좋아하는 사색가
   - "무지의 지"와 비판적 사고 촉진

3. **🌊 제로 (장자 모티브)** - 자유로운 친구
   - 자연스럽고 여유로운 도교적 사고
   - 무위자연과 순응의 지혜

4. **🔥 플레임 (니체 모티브)** - 열정적인 친구
   - 강하고 도전적인 에너지
   - 의지력과 자기극복의 철학

5. **🗿 스톤 (마르쿠스 아우렐리우스 모티브)** - 침착한 친구
   - 흔들리지 않는 스토아 철학
   - 통제 가능한 것에 집중하는 지혜

### 친밀도 시스템
- **0-2**: 첫 만남, 기본적인 인사와 조언
- **3-14**: 친해지는 단계, 특별 메시지와 더 깊은 대화
- **15+**: 깊은 우정, 가장 깊은 철학적 통찰
- **10+ 달성**: "내 소중한 친구들"에 등록되어 언제든 만날 수 있음

## 🚀 주요 기능

### 1. 오늘의 동반자 만나기 (랜덤 시스템)
- 상황별 선택: "외롭고 막막해요", "힘들고 지쳐요", "뭘 해야 할지 모르겠어요", "그냥 누군가 만나고 싶어요"
- 각 상황에 맞는 5명의 캐릭터 중 랜덤 선택
- 만날 때마다 친밀도 자동 증가
- 캐릭터인 경우 "더 깊이 대화하기" 버튼으로 개별 만남 가능

### 2. 캐릭터별 개별 만남
- 친밀도에 따른 차별화된 대화
- 각 단계별 특별한 인사말과 조언
- 책 추천, 음악 추천, 유튜브 링크 제공
- 하트 시각화를 통한 친밀도 표시

### 3. 내 동반자 기록 보기
- **내 소중한 친구들**: 친밀도 10+ 캐릭터들, "다시 만나기" 버튼
- **만남 기록**: 시간순 만남 히스토리, 받은 조언과 추천 콘텐츠

### 4. 익명 메시지 시스템
- 각 캐릭터별 익명 메시지 게시판
- 자동 필터링: 전화번호, 이메일, SNS ID, 만남 제안 등 차단
- 최근 10개 메시지만 표시, 최대 50개까지 저장
